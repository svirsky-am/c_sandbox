
# Задание 1. Серверы, клиенты и синхронизация в многопоточной системе
Вам предстоит написать многопоточную программу со множеством серверов и клиентов, в которой клиенты выступают акторами, а серверы — обработчиками событий. 
# Что нужно сделать
1. Реализовать серверы, которые будут:
- Разделять общий счётчик обращений.
- Изменять строку «эпохи».
- Работать в отдельных потоках.
2. Реализовать клиентов, которые могут:
- Увеличивать общий счётчик.
- Менять строку «эпохи».
- Порождать новые серверы.
- Завершать работу всех серверов.
3. Взаимодействие между клиентами и серверами должно происходить через каналы.
# Ожидаемый результат
1. Программа должна корректно обрабатывать команды:
- увеличение счётчика;
- изменение строки эпохи;
- порождение новых серверов;
- завершение работы всех серверов.
2. Серверы должны обновлять своё состояние (счётчик и эпоху) по мере получения команд.
